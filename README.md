# 🐍 Capstone Project – Python Fundamentals | Python

## 📝 Project Description
This capstone project demonstrates the application of **Python fundamentals, Pandas, and NumPy** to solve real-world data manipulation and analysis tasks.  
The project involves working with multiple datasets (Projects, Employees, Seniority Levels), performing **ETL operations, data cleaning, transformations, joins, and business rule automation**.  
The goal was to showcase proficiency in Python for data wrangling, logic building, and reporting workflows.

---

## 🔎 Overview
- Created three separate DataFrames (Projects, Employees, Seniority Levels) and saved them as `.csv` files.  
- Cleaned missing values in project costs using **running averages with loops**.  
- Transformed employee names into **First Name** and **Last Name** columns.  
- Joined all three DataFrames into a single **Final DataFrame**.  
- Applied business rules for promotions, demotions, and bonuses based on project outcomes.  
- Generated new DataFrames for aggregated project costs and employee-specific insights.  

---

## ✨ Key Features
- **Data Cleaning**: Handling missing values with running averages.  
- **Data Transformation**: Splitting names, adding prefixes (Mr./Mrs.), dropping unnecessary columns.  
- **Business Logic Automation**:  
  - Bonuses for employees with finished projects.  
  - Demotions for failed projects.  
  - Promotions for employees above a certain age.  
- **Filtering & Querying**: Extracting employees based on city names containing specific letters.  
- **Aggregations**: Calculating total project costs per employee.  

---

## 🛠️ Technology Used
- **Python**: Core programming language.
